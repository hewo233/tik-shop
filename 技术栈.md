### 字节青训营-微服务电商服务系统

`Cloudwego` `Kitex` `Hertz` `微服务` `IDL` `Swagger` `etcd` `Gorm` `PostgreSQL` `Thrift` `PASETO` `Go`

- **项目描述**：
  
  基于 CloudWeGo 生态实现的电商服务系统，具有用户管理、商品管理、订单管理、购物车等功能。项目使用了 Kitex + Hertz 实现微服务架构，etcd 作为服务注册中心，将用户服务、商品服务、订单服务、购物车服务等不同服务分离部署。使用 IDL (Thrift) 定义接口，团队协作实现设计的接口。采用 Gorm + PostgreSQL 进行现代化数据库管理，Swagger 提供 API 文档支持，PASETO 实现安全的身份认证。

- **项目能力**：
  - **企业级微服务架构设计与实践**：按照业务领域合理拆分服务，使用 IDL 生成代码，便于修改和移植，实现了服务的高内聚、低耦合
  - **高性能分布式系统开发**：采用字节跳动 CloudWeGo 生态（Kitex、Hertz、Sonic），实现高性能 RPC 通信和 HTTP 服务，掌握服务注册发现、负载均衡等分布式系统核心技术
  - **现代化工程实践能力**：使用 IDL 驱动开发确保接口契约一致性，通过代码生成提升开发效率，使用 Swagger 自动生成 API 文档，展现良好的工程化能力
  - **数据库设计与优化**：使用 GORM + PostgreSQL 进行数据持久化，支持读写分离、事务管理等高级特性，使用代码生成保证类型安全
  - **安全性设计意识**：采用 PASETO 现代化认证方案，规范错误码设计，独立配置管理，体现安全开发意识
  - **团队协作开发经验**：多人后端共同开发、维护项目，使用 Git 进行版本管理，掌握团队协作流程和规范
  - **全栈技术视野**：从网关层到微服务层，从数据库到缓存，从认证到API文档，具备完整的后端技术栈实践经验

## 详细技术栈

### 核心框架
- **CloudWeGo Kitex**：字节跳动开源的高性能 RPC 框架，用于微服务间通信
- **CloudWeGo Hertz**：字节跳动开源的 HTTP 框架，用于网关层 API 服务
- **Go 1.22.9**：主要开发语言，云原生时代首选

### 服务治理
- **etcd**：分布式键值存储，用于服务注册与发现
- **Apache Thrift**：IDL 定义语言，用于定义 RPC 服务接口和数据结构

### 数据层
- **PostgreSQL**：主要关系型数据库
- **GORM**：Go 语言 ORM 框架，包含代码生成、读写分离等高级特性

### 安全与认证
- **PASETO**：平台无关的安全令牌协议，比 JWT 更安全

### 工具链
- **Swagger/OpenAPI**：API 文档自动生成
- **Sonic**：字节跳动开源的高性能 JSON 序列化库
- **Viper**：配置管理
- **Makefile**：项目构建和代码生成自动化

### 微服务列表
1. **user-service**：用户管理服务
2. **product-service**：商品管理服务
3. **order-service**：订单管理服务
4. **cart-service**：购物车服务
